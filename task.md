```
Написать бота, который выполняет сканирование директории и сохраняет следующую 
информацию: 
- дерево директории, включая все файлы вложенные в поддиректории, т.е. Рекурсивно 
проходится по всему дереву; 
- все возможные атрибуты файла (размер, расширение, владельца, права доступа и т. д.). 
Также требуется иметь возможность сохранять историю изменения в директории, т.е. бот 
периодически сканирует и сохраняет информацию: период сканирование должен быть задан 
через конфигурацию. 
Обязательные требования: 
1. конфигурация бота: 
◦ директория сканирования (родительская директория); 
◦ периодичность сканирования; 
◦ максимальный уровень вложенности (параметр отвечающий на сколько глубоко 
необходимо сканировать по поддиректориям); 
◦ игнорируемое расширение (параметр необходимый для ограничения того, что не 
требуется сохранять). 
2. логирование работы бота; 
3. зафиксированные в файле зависимости; 
4. инструкция по запуску; 
5. код должен быть выложен на git. 
Дополнительным плюсом будет реализация следующего: 
1. возможность сохранять историю в любую СУБД (PostgreSQL, MariaDB, Redis, Tarantool, 
либо на ваш выбор); 
2. организация буфера для записи в БД, т.е. боты отправляют данные некоторому сервису, 
который по накоплению данных, записывает их в БД; 
3. все должно работать в docker; 
4. с помощью docker-compose организовать работу 3+ ботов с различной конфигурацией.
```
